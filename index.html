Direction

Deep Learning, Long Contexts, and LLM-Based Agents.

I am a Ph.D. student at George Mason University, advised by Prof. Xuan Wang. My current interests include deep learning, long-context and sequence modeling, LLM-based agents, agent memory and reliability, and evaluation methods that separate real capability from measurement artifacts.

I focus on problems that connect model mechanisms, implementation details, and empirical evaluation: how models generalize to longer inputs, aggregate information, reveal benchmark artifacts, and use memory safely in agent workflows.
